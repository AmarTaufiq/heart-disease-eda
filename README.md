# Heart Disease EDA Project

## Overview
This project focuses on Exploratory Data Analysis (EDA) of a heart disease dataset
to identify patterns, trends, and potential risk factors.

# Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

# Key Analysis
- Distribution of age, cholesterol, and blood pressure
- Relationship between heart disease and chest pain types
- Correlation analysis between numerical features

# Dataset
Source: Kaggle (Heart Disease Dataset)

# How to View
Open the `.ipynb` file directly on GitHub to view the full analysis and outputs.
